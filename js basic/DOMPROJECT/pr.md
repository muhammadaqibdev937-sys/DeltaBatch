### Projects:
  Topics the project covered: 

* **Fetching elements:** `getElementById`, `querySelector`, `getElementsByClassName`
* **DOM traversal:** `parentNode`, `children`, `nextSibling`
* **Manipulating elements:** `innerHTML`, `textContent`, `style`, `classList`, `setAttribute`
* **Creating/removing elements:** `createElement`, `appendChild`, `removeChild`
* **Events:** `click`, `scroll`, `keypress`, `input`, `submit`, `mouseover`
* **Event bubbling & delegation**
* **Form handling & validation**



## **1. Todo List App **

**Concepts Covered:**

* DOM fetching & traversal (`getElementById`, `querySelector`)
* Adding/removing elements dynamically (`createElement`, `appendChild`, `removeChild`)
* Event handling (`click`, `keypress`)
* Styling elements with `classList`

**Extra Challenge:**

* Add **filtering tasks** (completed vs pending)
* Use **localStorage** to save tasks

---

## **2. Interactive Form Validator**

**Concepts Covered:**

* Forms & input handling (`onsubmit`, `onchange`, `preventDefault`)
* Validating input data (email, password, phone number)
* Showing dynamic error messages (`innerHTML` / `textContent`)

**Extra Challenge:**

* Add **live validation** while typing
* Highlight fields dynamically using `style` or `classList`

---

## **3. Image Gallery / Lightbox**

**Concepts Covered:**

* Creating/removing elements dynamically
* Handling click events to open/close images
* DOM traversal to find siblings/parents
* Animating classes with `classList.toggle`

**Extra Challenge:**

* Add **next/previous buttons** using `nextSibling` and `previousSibling`
* Lazy loading images on scroll (`scroll` event)

## **4. Custom Dropdown / Accordion**

**Concepts Covered:**

* Event delegation (handling clicks on multiple elements)
* Toggling classes (`classList.toggle`)
* Parent/child traversal (`parentNode`, `children`)
* Scroll/resize events (optional for animations)

**Extra Challenge:**

* Only one section opens at a time (accordion behavior)
* Close dropdown when clicking outside
